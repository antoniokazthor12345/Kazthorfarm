# KAZTHOR FARM

Project evolution and future development plan.

---

# 🎯 Vision

KAZTHOR FARM aims to become a complete smart agriculture platform built around ESP32 devices, cloud communication, automation and AI-assisted monitoring.

The project is being developed incrementally, with each version introducing new hardware, software and automation capabilities.

---

# ✅ V1 — Core Farm System

Status:

```txt
Released
```

Main Features:

* ESP32-S3 Controller
* ESP-IDF Development
* FreeRTOS Architecture
* DS18B20 Temperature Monitoring
* Soil Moisture Monitoring
* Water Level Monitoring
* OLED Display
* MQTT Cloud Integration
* Real-Time Dashboard
* FAN Control
* MIST Control
* RGB Light Control
* Timer Control
* PC817 Optocoupler Interface
* Relay-Based Power Control
* Alert and Critical State Management

Goals Achieved:

* Stable sensor monitoring
* Cloud connectivity
* Climate automation
* Dashboard visualization
* Modular software architecture
